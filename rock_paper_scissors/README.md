# The Python Rock, Paper, Scissors Game

About: An interactive CLI rock, paper and scissors game where you play against the computer. 

## How to use 

Run `python main.py` and follow the prompts to input the relevant responses. 

## What it does

It is you vs the computer. Choose rock, paper or scissors against the computer and see how it goes!

### Learning outcomes 

- random library
- random.int
- if/elif/else
- ASCII art
