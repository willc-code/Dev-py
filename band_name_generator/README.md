# Band Name Generator

## About
A simple command line python app that generates the name of a band based on the user's input. The program asks the user two simple questions - the colour of the shirt that you are wearing, and the last food you ate - and combines them into a unique band name. 

## Features
- Interactive command line prompts. 
- Generates a band name based on the user's input. 

## Requirements
- Python 3.12+
